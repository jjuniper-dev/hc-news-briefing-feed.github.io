2) GREP-ExP: Canonical Definition

Full name: Global Repository of Epidemiological Parameters – Extraction of Parameters.

GREP-ExP is an LLM-powered systematic literature review pipeline jointly owned by EPSGH/SPIB (Waddell, Corrin, Pussegoda) and DAS/SDSE (Bing Hu et al.). It automates screening and extraction of epidemiological parameters for the WHO GREP project.

Architecture pattern

  • Screening Agent + Critical Agent + Ensemble Agent + human-in-the-loop feedback.
  • Uncertainty is managed using inter-agent disagreement, not self-confidence scores.
  • Published/accepted in Cochrane Evidence Synthesis and Methods (2025).
