Intelligence workflow

CLAUDE.md was not found in this repository, so this editor includes the required operational workflow directly.

1) Classify update type

2) Versioning and log

3) Required 7-step flow

  1. Draft in editor (markdown).
  2. Analyze with local LLM review.
  3. Publish to intelligence-draft branch.
  4. Update date, version, and change log.
  5. Check quality standards (clarity, evidence, traceability).
  6. Apply tone guidelines (briefing-ready, neutral, decision-oriented).
  7. Compare with similar examples before finalizing.

Examples in repo: content/assessments/cohere/v0.1/assessment.md and reviews/hc-pptx-workflow-assessment.md.
