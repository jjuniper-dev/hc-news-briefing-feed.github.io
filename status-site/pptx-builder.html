End-to-End Gates

Gate A — Brief Intake
Capture audience, objective, decision, timeframe, source tabs/ranges, and deadline.
Gate B — Insight Contract
Require KPI values, trend direction, outlier note, source ranges, and known limitations.
Gate C — Story Contract
Each slide needs a decision-oriented title, evidence, chart type, and action.
Gate D — Build Contract
Use HC master template, one core claim per slide, and source/date footers.
Gate E — QA Contract
Run content QA + visual QA + accessibility QA before sign-off.
